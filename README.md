# sql-data-analytics-project
A comprehensive collection of SQL scripts for data exploration, analytics, and reporting. These scripts cover various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more.

This repository was designed to quickly explore, segment, and analyze data within a relational database. Each script focuses on a specific analytical theme and demonstrates best practices for SQL queries.

This project was developed following the instructions of DataWithBaraa [Course Link](https://youtu.be/SSKVgrwhzus) | [GIT Repo](https://github.com/DataWithBaraa/sql-data-analytics-project).
